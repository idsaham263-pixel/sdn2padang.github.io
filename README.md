# Website SD Negeri 2 Padang

Repository ini berisi source code website resmi **SD Negeri 2 Padang**,  
Kecamatan **Tanggungharjo**, Kabupaten **Grobogan**, Jawa Tengah.

Website ini dibuat sebagai media:
- Informasi resmi sekolah
- Publikasi kegiatan dan prestasi
- Sarana komunikasi dengan orang tua dan masyarakat

## Teknologi
- HTML
- CSS
- GitHub Pages (hosting gratis)

## Tujuan
Memberikan akses informasi yang mudah, cepat, dan transparan kepada masyarakat
serta mendukung digitalisasi sekolah dasar.
